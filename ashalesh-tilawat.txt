    1. What is Semantic HTML? 

    Helps you give meaning to your code.
    For example, <article>, <form>, or <table>. 
    Each describe a specific type of content.

    2. What is HTML used for? 

    HTML is used to structure the bare bones of a webpage.
    It creates the foundation from which a webpage is constructed.

    3. What is an attribute and where do we put it? 

    Attributes provide additional info about HTML elements.
    For example, they can tell you where the element is located. 
    Using href or src, one can describe where an img is. 

    4. What is the h1 tag used for? How many times should I use it on a page?

    <h1> is used on the title of the webpage. 
    It should only be used once to highlight the most important title in a page.

    5. Name two tags that have required attributes

    <img> requires src. External links require hrefs.

    6. What do we put in the head of our HTML document? 

    <header>

    7. What is an id? 

    An attribute that identifies the element and separates it from the rest. 
    This identifier has to be unique to that element.

    8. What elements can I add an id to? 

    To all elements.

    9. How many times can I use the same id on a page? 

    Once.

    10. What is a class? 

    A class is used to identify a group of elements. 
    Then you can use the class to call that group of elements in CSS.

    11. What elements can I add a class to? 

    Any element you want. 

    12. How many times can I use the same class on a page? 

    As many times as you want. 

    13. How do I get my link to open in a new tab?

    href=... target="_blank"

    14. What is the alt attribute in the image tag used for? 

    To identify the source of the image.

    15. How do I reference an id?

    #.

    16. What is the difference between a section and a div

    <Section> groups together the elements within it. 
    It designates a portion of the code as a section. 
    <Div> has no special meaning, but it can be used to parse the code. 

    17. What is CSS used for? 

    To style the HTML. 

    18. How to we select an element? Example - every h2 on the page

    19. What is the difference between a class and an id? - Give me an example of when I might use each one

    An ID helps you pick out one specific element in the sea of code. 
    Classes identify a group of elements. Can be used to pick out that group of elements from the rest.

    22. What is a parent child selector? When would this be useful?

           Your name should be listed before the slash.

        4) Copy the link in the menu. 
        5) Open documents/git/ using Git Bash.
        6) Use git clone and paste the link you copied. 
        7) Use cd to move into the new repository. 
        8) Use git checkout -b 'ashalesh-tilawat' to create your own branch. 
        9) Use code . to edit the code in VS code. 
        10) Use git commit to commit the changes you have made.
